Page yang ada disini :

2. Halama Login
3. Halaman Dashboard (menampilkan data post)
4. Show comment
5. Halaman profile

# Getting Started

1. Silahkan clone project
2. Jalankan "npm install"
3. Masuk kedalam project di local komputer
4. Jalankan npm start
5. Pastikan internet terhubung
